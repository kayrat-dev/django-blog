Блог на Django 📝

Привет! Это мой проект — блог, написанный с нуля на Django.

Хотел разобраться как всё устроено под капотом — модели, представления, маршрутизация, база данных. Этот проект и стал результатом.

Что работает


Пагинация — посты разбиты на страницы, никакой бесконечной ленты
Канонические URL — чистые и читаемые ссылки для каждого поста
PostgreSQL готов — локально использую SQLite, но dj-database-url уже настроен для продакшена
Админка — настроена для управления постами без правок в коде


Стек


Python 3.x
Django
SQLite (разработка) / PostgreSQL (готово к деплою)
